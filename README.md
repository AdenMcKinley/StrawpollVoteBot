## StrawpollVoteBot ##

This is a [Strawpoll](http://strawpoll.me/) Vote Bot

## Usage ##




**Update the proxy list**


    php update_proxies.php

----------

**Run the Vote Bot**


    php vote.php <VoteID> <Option> <Amount>

    For example:
    php vote.php 3491291 2 100
